viewglob 1.0 released
    sjbach - 2004-09-30 06:11   -   viewglob
viewglob is a tool to increase the usability of the Unix shell by leveraging the expressiveness of graphical environments. It sits as a layer beneath an xterm and watches your bash or zsh shell activity as you type. An interactive GTK+ display shows the layout of relevant directories and highlights file selections and potential name completions. This 1.0 release makes several stability fixes and adds two useful features: 1. the ability to double click a file or directory in the display and have its name or path sent to the terminal; 2. the ability to type into the display and have your keystrokes entered into the terminal.

SSL-Explorer v0.1.1 released!
    rpernavas - 2004-09-30 06:11   -   SSL-Explorer
The 3SP SSL-Explorer is the world's first open-source SSL-based VPN solution of its kind. Released under the GPL, this entry-level VPN solution provides users with a means of securely managing their IT infrastructure from outside the network perimeter, armed with just a standard web browser. This second release now provides full Active Directory authentication for use within Microsoft Windows Server environments. The release also sees a change from using the MySQL database to an internal HSQLDB database and this means that post-installation database configuration is no longer required.
